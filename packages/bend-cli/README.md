Global CLI wrapper for Bend - the production-ready backend scaffolder.

## Installation

```bash
npm install -g bendjs
```

## Usage

After global installation:

```bash
bend
```

## Why This Package?

This is a thin wrapper around `bend-core` for users who prefer global CLI installation. Most users should use `create-bend` instead:

```bash
npm create bend@latest
```
